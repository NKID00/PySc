## 优点
- 无需使用改版Scratch
- 只需一次运行，自动开机启动
- 几乎不耗性能
- 项目以GPL开源

## 目标
- 无需安装Python
- 支持 `time.sleep()`
- 可以在Scratch上一次性执行多行Python脚本
- 移植到linux和MacOSX上
